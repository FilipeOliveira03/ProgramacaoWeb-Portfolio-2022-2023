Projetos Realizados


O meu primeiro projeto foi um pequeno minesweeper, realizado por mim para a cadeira de fundamentos de programação do 1º ano, do 1º semestre do curso, no ano de 2021. Foi realizado em Kotlin e apenas foi usado o básico da programação. Na altura, não tinha conhecimento do GitHub, por isso não tenho o projeto em mão para poder disponibilizar para poderem ver.

No 2º semestre, ainda no 1º ano (já em 2022), realizei um projeto em java para a cadeira de Algoritmia e Estruturas de Dados, que era uma organização de vários ficheiros, cheios de dados, tipo uma base de dados. Com os ficheiros organizados, realizava-mos queries para poder retirar informações dos mesmos dados. Foi o meu primeiro projeto em java e acontece o mesmo que o de Kotlin, não tinha acesso ao git e não tenho como mostar o que fiz.

Na mesma altura, ou seja, no 2º semestre do 1º ano, tambem realizei um projeto em C para a cadeira de Linguagens de Programação 1, que era um pequeno jogo tabuleiro que era tipo uma "corrida", que, jogando apenas com 2 jogadores e 4 peões cada, se um dos peos de um dos jogadores passase outro de outro jogador, este seria eliminado. Haviam casas que, se o jogador fosse passado nessa mesma casa, ele não perderia. Perdia quem ficasse sem peões no final do jogo. Mais uma vez sem conhecimento do git, não consigo mostrar.

O mais recente, realizado no 1º semestre do 2º ano no ano de 2022, realizei eu e mais um colega meu, um projeto em Java para a cadeira de Linguagens de Programação 2, que era um jogo de tabuleiro, com um visualizador gráfico, de 2 a 4 jogadores, em que o objetivo era chegar a casa final. O jogo continha diferentes "espécies", em que cada uma se movia de maneiras diferentes no tabuleiro. Espalhado pelo mapa, haviam alimentos que cada jogador tinha que ingerir e aconteceriam difrentes reações ao alimento. Este projeto está disponivel no git, clicando aqui.

Tambem no 1º semestre do 2º ano no ano, realizei um projeto para a cadeira de Arquiteturas Avançadas de Computadores, em que consistia em um prototipo de um alarme com um código para poder desbloquear uma porta, em que o utilizador, com 3 tentativas, tinha que inserir um código de 4 digitos. Se acertasse, a porta abria, mas no entanto, se o utilizador falhasse as 3 vezes, o alarme iria tocar até ser desligado. Está disponivel um pequeno video no youtube, clicando aqui.

Para a cadeira de programação web, do 2º ano do 2º semestre, realizei alguns laboratórios, sendo os próprios: LAb2, LAB3 e LAB4.




Tecnologias usadas neste Website

HyperText Markup Language, mais conhecido por HTML, foi criado em 1991 por Timothy John Berners-Lee. HTML é uma linguagem de marcação utilizada na construção de páginas na Web e os seus documentos HTML podem ser interpretados por navegadores. Nesta página foi usado para, principalmente para a colocação de texto e imagens na página.

Cascading Style Sheets, mais conhecido por CSS, foi cridado em 1996 por World Wide Web Consortium, a principal organização da World Wide Web (www). CSS é um mecanismo para adicionar estilos a uma página web. Nesta página, foi usado para isso mesmo.

JavaScript, tambem conhecido por JS, surgiu em 1995, criado por Brendan Eich. JavaScript é uma linguagem de programação interpretada estruturada, de script em alto nível com tipagem dinâmica fraca e multiparadigma, sendo uma das três principais tecnologias da World Wide Web. Foi usada para poder fazer o butão de dark mode, animações, etc
